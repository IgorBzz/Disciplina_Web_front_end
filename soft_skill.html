Introdução ao HTML    

O que é HTML?

   

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Com HTML, é possível estruturar textos, imagens, links e outros conteúdos visuais de uma página.

Principais elementos do HTML:

   
           
  • <h1> a <h6>: Títulos do conteúdo.
    •        
  • <p>: Parágrafos de texto.
    •        
  • <a>: Links para outras páginas.
    •        
  • <img>: Imagens dentro da página.
    •        
  • <ul>, <ol>, <li>: Listas não ordenadas, ordenadas e seus itens.
    •

  1. Usar indentação para organizar o código.

  2. Escrever tags corretamente (abrir e fechar).
    3.        
  3. Usar comentários para explicar partes do código.

  4. Manter o código limpo e legível.
